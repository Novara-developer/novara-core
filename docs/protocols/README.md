# Novara protocols docs v0.1

ここは **「コードに入る前の、人間向けプロトコルだけ」** を置く場所。

- 事故が起きたときに、誰がどう動くか
- 被害者や一般の人に、どう説明するか
- 憲法やルールをどうやって変えていくか

を **文章だけで決めるレイヤ**。

v0.1 では、まだ **outline（骨組み）** だけを置いている。

---

## ファイル一覧（v0.1）

- `novara-incident-protocol-outline-v0.1-ja.md`  
  AI やシステムがやらかしたときの「事故対応プロトコル」の骨組み。  
  どのログを凍結するか、誰が最初に判断するか、72h 以内に何を公開するか、など。

- `novara-proof-rail-v0.1-spec.md`  
  お金や権限を動かす前に **「証拠レールを通す」** ための最低ライン。  
  「証拠バンドルが OK じゃないと支払いを流さない」というルールの、テキスト版の v0.1。

- `novara-constitution-change-process-outline-v0.1-ja.md`  
  Novara 憲法や core 原則を **どうやって変えるか** の手続き。  
  何人のガーディアンが必要か、緊急パッチと通常改定の違いは何か、など。

---

## v0.1 の位置づけ

- どれも **「outline」＝骨だけ** で、まだ最終版ではない  
- 完全な v1.0 spec は、将来 `Novara` リポジトリ側（Evidence OS 側）に  
  - JSON Schema / API / CLI の仕様と一緒に固めていく予定

ここはあくまで

- 「2040 までに絶対に崩したくない筋」
- 「資本・国家・ASI に書き換えられたくないルール」

を **人間語で先に決めておく保険レイヤ**。
